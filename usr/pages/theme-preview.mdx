# Academic Theme Preview

Welcome to the enhanced **s:CMS Academic Theme**! This page showcases the modern, clean design with academic-friendly typography and professional color palette.

## Typography Showcase

### Headings Hierarchy
The headings use **Playfair Display** - an elegant serif font that brings sophistication while maintaining excellent readability:

# Main Heading (H1)
Featuring gradient text effect and optimized spacing for maximum impact.

## Section Heading (H2) 
Clean, professional styling with elegant serif typography for major sections.

### Subsection Heading (H3)
Perfect for detailed content organization with refined styling.

#### Minor Heading (H4)
For fine-grained content structuring with consistent hierarchy.

### Body Text
Body text now uses **Inter** - a modern, highly readable sans-serif font designed specifically for digital interfaces. At 1.125rem base size with improved line-height, it ensures excellent readability across all devices while maintaining a contemporary, professional appearance perfect for academic content.

> **Blockquotes** are styled with modern borders and typography to highlight important information or citations in academic context.

## Code Display

Code blocks use **JetBrains Mono** for optimal readability:

```javascript
// Modern academic theme configuration
const academicTheme = {
  primary: '#1e3a8a',
  secondary: '#64748b', 
  accent: '#d97706',
  fonts: {
    heading: 'Playfair Display',
    body: 'Inter',
    mono: 'JetBrains Mono'
  },
  typography: {
    baseSize: '1.125rem',
    lineHeight: 1.7
  }
};
```

Inline code like `fetch()` and `const myVariable = 'value'` maintains readability within paragraphs.

## Design Principles

This academic theme follows key design principles:

1. **Readability First**: Typography choices optimize for extended reading sessions
2. **Professional Aesthetics**: Color palette suitable for academic institutions
3. **Modern Interactions**: Subtle animations and hover effects enhance user experience
4. **Accessibility**: High contrast ratios and clear visual hierarchy
5. **Responsive Design**: Seamless experience across all device sizes

The theme maintains the separation between core functionality and user customization, allowing for easy updates while preserving institutional branding and custom styling preferences.
