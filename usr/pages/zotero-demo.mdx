## Usage Examples

### Basic Usage
```jsx
import { ZoteroGeoViewerMdx } from '@core/components/ZoteroGeoViewer';

<ZoteroGeoViewerMdx groupId={12345} />
```

### Custom Configuration
```jsx
<ZoteroGeoViewerMdx 
  groupId={12345}
  layout="6x6"
  mapHeight="400px"
  mapCenter="12.5,42.0,6"
  maxItems={500}
  tagAutocomplete={false}
/>
```

### In Astro Files
```astro
---
import { ZoteroGeoViewer } from '@core/components/ZoteroGeoViewer';
---

<ZoteroGeoViewer 
  groupId={12345}
  layout="8x4"
  client:idle
/>
```

## Technical Details

### Data Sources
- **Zotero API**: `https://api.zotero.org/groups/{groupId}/items`
- **Coordinate Data**: `/data/zoteroTagCoordinates.geojson`
- **Matching**: Items matched to locations via tag names

### Performance
- Lazy loading with `client:idle`
- Client-side filtering for responsive search
- Optimized GeoJSON rendering
- Efficient tag autocomplete

### Dependencies
- React with hooks for state management
- MapLibre GL for map rendering  
- Tailwind CSS for styling
- Zotero public API for data
