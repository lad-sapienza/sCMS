## Component Features

The ZoteroGeoViewer component provides:

### 📍 Geographical Visualization
- Interactive map with location markers sized by item count
- Popup windows showing item details and links
- Automatic map bounds fitting to data

### 🏷️ Tag-based Filtering  
- Real-time search through Zotero tags
- Autocomplete suggestions for easy discovery
- Client-side filtering for fast interaction

### 📊 Live Statistics
- Total items in the library
- Georeferenced items count  
- Number of unique locations
- Updates automatically as you filter

### 📚 Recent Items Preview
- Shows latest additions to the library
- Direct links to Zotero items
- Author and publication info

## Usage Examples

### Basic Usage
```jsx
import { ZoteroGeoViewer } from '@core/components/ZoteroGeoViewer';

<ZoteroGeoViewer groupId={12345} client:load />
```

### Custom Configuration
```jsx
<ZoteroGeoViewer 
  groupId={12345}
  layout="6x6"
  mapHeight="400px"
  mapCenter="12.5,42.0,6"
  maxItems={500}
  tagAutocomplete={false}
  client:load
/>
```

### In Astro Files
```astro
---
import { ZoteroGeoViewer } from '@core/components/ZoteroGeoViewer';
---

<ZoteroGeoViewer 
  groupId={12345}
  layout="8x4"
  client:idle
/>
```

## Technical Details

### Data Sources
- **Zotero API**: `https://api.zotero.org/groups/{groupId}/items`
- **Coordinate Data**: `/data/zoteroTagCoordinates.geojson`
- **Matching**: Items matched to locations via tag names

### Performance
- Lazy loading with `client:idle`
- Client-side filtering for responsive search
- Optimized GeoJSON rendering
- Efficient tag autocomplete

### Dependencies
- React with hooks for state management
- MapLibre GL for map rendering  
- Tailwind CSS for styling
- Zotero public API for data
