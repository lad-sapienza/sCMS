# Use the `<Search />` component to connect to the Zotero API

This example shows how to use the `<Search />` component to query Zotero's API. It supports searching by title, tags, and other fields available in Zotero.

<div className="row">
  <div className="col-sm-4">
```jsx
  <Search
    source={{ customApi: ZoteroService }}
    fieldList={{
      q: "Search Query",
      tag: "Tags",
      title: "Title",
      bookTitle: "Book Title",
      creatorSummary: "Author(s)",
      date: "Publication Date",
    }}
    resultItemTemplate={(item) => (
      <div className="card my-3" key={item.key}>
        <div className="card-body">
          <h5 className="card-title">{item.title}</h5>
          <p><strong>Author(s):</strong> {item.creatorSummary}</p>
          <p><strong>Publication Date:</strong> {item.date}</p>
          <p><strong>Book Title:</strong> {item.bookTitle}</p>
      
          <a href={item.links?.alternate?.href} target="_blank" rel="noopener noreferrer" className="btn btn-primary">
            View Details
          </a>
        </div>
      </div>
    )}
  />
```
