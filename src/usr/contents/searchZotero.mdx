# Use the `<Search />` component to connect to the Zotero API

This example shows how to use the `<Search />` component to query Zotero's API. It supports searching by title, tags, and other fields available in Zotero.

<div className="row">
  <div className="col-sm-4">
```jsx
     <Search
  limitTo="advanced"
  source={{ customApi: ZoteroService }}
  operators={{"_contains": "Contains"}}
  fieldList={{
    q: "Search Query",
    tag: "Tags",
  }}
  resultItemTemplate={(item) => {

    return item ? (
      <div key={item.key} className="card my-3">
        <div className="card-body">
          <h5 className="card-title">{item.title}</h5>
          <p>
            <strong>Authors:</strong> {item.authors}
          </p>
          <p>
            <strong>Tags:</strong> {item.tags}
          </p>
          <p><strong>Publication Date:</strong> {item.date}</p>

          <a href={item.links?.alternate?.href} target="_blank" rel="noopener noreferrer">
            More Details
          </a>
        </div>
      </div>
    ) : (
      <div className="text-muted">No results found.</div>
    );
  }}
/>

```
