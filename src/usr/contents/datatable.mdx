# Build customisable tables with data from your database or static files

`<DataTb />` component can be used in MDX pages to display a dynamic, fully customisable table of data from different sources, such as a static CSV file, both hosted locally or online, or a [Directus API](https://docs.directus.io/reference/introduction.html).

Under the hood, `DataTb` uses the [React Data Table Component](https://react-data-table-component.netlify.app/?path=/docs/getting-started-intro--docs).


## Example of a Directus instance as data source
<div className="row">
  <div className="col-sm-8">
    <DataTb
      source={{
        directus: {
          table: "scms_ksa",
          queryString: "limit=1000&offset=0",
        }
      }}
      striped={true}
      columns={[
    {
      name: "Label",
      sortable: true,
      cell: item => (
            <a href={`../record/?tb=scms_ksa&id=${item.id}`}>
              {item["Item_Label"]} 
            </a>
          ),
    },
    {
      name: "Site name",
      selector: row => row["Site_Name"],
      sortable: true,
    },
    {
      name: "Thumbnail",
      cell: item => (
            <img src={item["Thumbnail"]} />
          ),
      sortable: true,
    },
    {
      name: "Date",
      selector: row => `${row["Early"]} / ${row["Late"]}`,
      sortable: true,
    },
    {
      name: "Collection summary",
      selector: row => row["Collection_Summary"],
      sortable: true,
    },
    
  ]}
    />
  </div>

  <div className="col-sm-4">
