#### `source`
- Object
- Required
- Default: _null_

An object containing information to source data. This should include the necessary properties for querying the data source.
Data can be sourced from a static file (typically a CSV file stored locally or available on the web), a Directus API, or a custom API.

If a static file is used, the `source` object should contain the following properties:
```jsx
{
  path2data: {
    path: "path/to/data.csv"
  }
}
```

Id a Directus API is used, the `source` object should contain the following properties:
```jsx
{
  directus: {
    /**
     * Directus endpoint. It can be omitted if env variable GATSBY_DIRECTUS_ENDPOINT is set.
     */
    endPoint: "URL to Directus instance",
  
    /**
     * Directus table name.
     * Required
     */
    table: "Directus referenced table name",
    
    /**
     * Directus optional filters and other, provided as querystring compatible to Directus API
     * Optional
     */
    queryString: "Some filters etc., compliant to Directus API",
    
    /**
     * Directus access token. It can be omitted if env variable GATSBY_DIRECTUS_TOKEN is set.
     */
    token: PropTypes.string,
    
    /**
     * Id of a specific record to retrieve
     */
    id: PropTypes.number,

    /**
     * Field that contains geographical data to display on map
     */
    geoField:PropTypes.string,
  }
}
```

Finally, if a custom API is used, the `source` object should contain the following properties:
```jsx
{
  customApi: {
    /**
     * Function to format the URL
     * Required
     */
    formatUrl: (searchString) => {
      return `https://api.example.com/search?q=${searchString}`;
    },
    
    /**
     * Function to parse the response
     * Required
     */
    parseResponse: (response) => {
      return response.data;
    }
  }
}
```
